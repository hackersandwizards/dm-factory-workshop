# dm-factory-workshop

## Setup

Teilnehmende brauchen:

- Claude Code
- `beans` CLI
- `jq`
- Node.js >= 20
- `npm` für Dependencies installieren, TypeScript bauen und Tests ausführen
- `git`

Beans CLI:

```bash
brew install hmans/beans/beans
beans version
```

Sandbox bauen:

```bash
cd sandbox
npm install
npm run build
npm test
```
